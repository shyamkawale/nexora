# How to Send Messages

## One-on-One Messaging

**Sending a Private Message**

1. Find the person you want to message
2. Click to open the chat window
3. Type your message
4. Hit Enter or click Send
5. Your message appears instantly!

**What You Can Do in Direct Messages**
- Send text messages
- Share files (documents, photos, videos)
- See when your message was delivered
- Know when they're typing
- Search old conversations
- Delete messages
- Pin important messages

## Group Chat

**Creating a Group**

1. Click "New Group" or "Create Group"
2. Give it a fun name (like "Marketing Team" or "Project Alpha")
3. Select people to add
4. Click "Create"
5. Start chatting!

**Group Chat Features**
- Chat with multiple people at once
- Add or remove members anytime
- See full message history
- Get notifications for new messages
- See who's typing

## Messaging Tips

**For One-on-One Chats**
- Use for personal or sensitive topics
- Perfect for quick questions
- Good for focused discussions

**For Group Chats**
- Great for team projects
- Keep everyone in the loop
- Perfect for brainstorming
- Share updates with multiple people

## Organizing Your Chats

**Chat Management**
- Pin important chats to top
- Archive old conversations
- Mute notifications when needed
- Search for specific chats

**Search in Messages**
1. Click the search icon
2. Type keywords
3. Find the message you need

## Message Notifications

**Stay Updated**
- Get alerts for new messages
- See notification count
- Customize notification settings
- Mute specific chats

**Notification Settings**
- Turn on/off notifications
- Choose sound/vibration
- Set quiet hours
- Manage notification preview

## Sharing Files in Chat

**Share in Messages**
1. Click the attachment icon (📎)
2. Choose your file
3. Add a message
4. Click Send
5. Recipient downloads anytime

**File Security**
- Only recipients can see files
- Files stay private
- Can share multiple files
- Can delete shared files

- Don't share sensitive info

**Stay Professional**

## Troubleshooting

**Messages Not Sending**
- Check your internet connection
- Try refreshing the app
- Check if recipient exists
- Try reloading the page

**Can't See Messages**
- Verify internet connection
- Refresh the conversation
- Log out and back in
- Check privacy settings

**File Sharing Issues**
- Check file size (max 50MB)
- Verify file type is supported
- Check your internet
- Try different browser

**Notifications Not Working**
- Check notification settings
- Verify permissions in settings
- Turn off/on notifications
- Restart the app
