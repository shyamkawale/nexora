# Uploading and Sharing Files

## Uploading a File

**Where Can You Upload?**

In a **Private Message**:
1. Click the attachment icon (📎)
2. Choose your file
3. Click Send
4. Your friend gets the file instantly

In a **Group Chat**:
1. Click the attachment icon (📎)
2. Choose your file
3. Add a message if you want
4. Click Send
5. Everyone in the group sees the file

In a **Post**:
1. Create your post
2. Click the attachment icon (📎)
3. Choose one or more files
4. Write your post text
5. Click Post
6. Everyone on your feed sees it

## File Size & Types

**Maximum File Size**: 50MB per file

**Supported File Types**:
- **Documents**: PDF, Word (.doc, .docx), Excel (.xls, .xlsx), PowerPoint (.ppt, .pptx), Text (.txt)
- **Images**: JPG, PNG, GIF, WebP
- **Videos**: MP4, MOV, AVI, WebM
- **Archives**: ZIP, RAR
- **Other**: JSON, CSV, XML

## Downloading Files

**How to Download**:
1. Find the file in a message or post
2. Click the download arrow (⬇️)
3. Choose where to save on your computer
4. Open when ready

**File Access**:
- Only people with access can download
- Private messages = only you and recipient
- Group chats = only group members
- Posts = depends on your privacy settings

## Sharing Files

**In Conversation**:
- Upload in a message
- Recipient gets instant notification
- Can download anytime

**In Posts**:
- Share with wider audience
- Great for sharing important documents
- Can attach multiple files

**Tips for Sharing**:
- Name files clearly (not "Document1.pdf")
- Include context in your message
- Check file size before uploading

## File Security

**Your Files Are Safe**:
✓ Only shared with people you want  
✓ Private by default  
✓ Encrypted during transfer  
✓ Secure download links

**Best Practices**:
- Don't share sensitive information in public posts
- Use private messages for confidential files
- Remove access if person leaves group/team
- Delete old files you no longer need

## Troubleshooting

### Upload Fails
- Check file size (max 50MB)
- Verify file type is supported
- Check internet connection
- Try refreshing the page

### Can't Download a File
- File may have been deleted
- Check your internet connection
- Try a different browser
- Contact the person who shared it

### Slow Upload/Download
- Check your internet speed
- Close other apps using network
- Try uploading smaller files first
- Contact support if persistent

## Tips for Better File Management

**Organize**:
- Name files descriptively
- Use folders when possible
- Archive old conversations

**Clean Up**:
- Delete files you no longer need
- Remove access to shared files when appropriate
- Check storage regularly

**Collaborate**:
- Share documents with team members
- Use comments to discuss files
- Version control by adding dates to filenames (report_jan2024.pdf)
